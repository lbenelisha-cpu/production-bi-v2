# ADAMA Production BI

פרויקט React + Vite + PWA עבור מערכת BI לניהול ייצור, אריזה, איכות, יעדים ודוחות.

## הגדרות Netlify

Build command:
npm run build

Publish directory:
dist

## מה קיים בגרסה זו

- React + Vite
- PWA בסיסי
- תפריט צד
- Dashboard ראשי
- מסך הגדרות יעדים
- יעדי מתקנים ניתנים לעריכה
- מתקן 1521 נוסף
- היסטוריית שינויי יעדים
- בסיס למודולי איכות, ייצור, דוחות ו־WhatsApp
